# 声绘 Google Play 发布准备清单

## 已在 App 内覆盖

- 应用名：声绘。
- 包名：`com.yunque.shenghui`。
- `targetSdk`：35，满足 Android 15 / API 35 目标版本要求。
- 权限说明：设置中心的“隐私与权限”页面说明了录音、文件、本地存储、API Key 本地保存和网络传输边界。
- 用户授权确认：创建音色前要求用户确认拥有声音和文本的合法授权，并提示不得克隆、冒用或传播未经授权的他人声音。
- 禁止行为提示：设置中心的“版权与授权声明”页面列出未授权克隆、冒充、诈骗、骚扰、虚假宣传等禁止行为。
- 远程配置预留：已预留广告位、弹窗通知、推广链接、版本更新、强制更新和更新地址的 Remote Config key。

## Google Play Data Safety 建议填写

实际提交前要以最终功能为准复核。当前版本建议按以下口径准备：

- 收集的数据：用户主动输入的文本、表演指令、上传或录制的参考音频、生成音频、API Key 配置。
- 数据用途：语音生成、文本优化、音色克隆、历史记录、本地播放、下载和分享。
- 数据传输：调用用户配置的语音服务或文本优化服务时，相关文本、指令或参考音频会发送到该服务接口。
- 本地存储：生成音频、历史记录、用户创建的音色、参考音频副本、API Key 配置保存在本机。
- 第三方共享：除用户主动配置并调用的语音服务、文本优化服务和主动分享动作外，当前版本不主动向声绘后台上传 API Key 或历史记录。
- 删除方式：用户可删除历史记录和自定义音色；卸载 App 会清除本地数据。

## 隐私政策草稿要点

正式上架需要一个可公开访问的隐私政策 URL。建议包含：

- 开发者名称、联系方式和适用范围。
- 麦克风权限用途：仅用于用户主动录制参考音频。
- 文件访问用途：仅用于用户主动选择音频或导入文档文本。
- 本地存储用途：保存历史、生成音频、自定义音色、参考音频和配置。
- API Key 说明：仅保存在用户设备本地，不上传到声绘后台。
- 第三方服务说明：用户配置的语音服务和文本优化服务会接收必要请求数据。
- 用户权利：删除历史记录、自定义音色、本地文件和卸载清除数据。
- 未成年人和禁止用途说明。
- AI 生成内容反馈与处理方式：当前 App 内不保留举报入口，正式发布前需确定可实际处理的反馈渠道。

## 仍需产品/后台配合

- 正式签名、Play App Signing 和上传密钥。
- Firebase 项目配置文件：`google-services.json`，用于 Remote Config / FCM。
- AI 生成内容反馈渠道：如果后续上架平台要求应用内入口，需要先准备可实际处理的后台、邮箱或客服系统。
- 公开隐私政策网页 URL。
- Google Play 后台的 Data safety 表单、AI-generated content 表单和应用内容评级问卷。

## 依赖生态跟进

- 已升级 `file_picker` 到 12 beta 线并迁移到新的静态 API，解决当前项目里 `file_picker` 的旧 KGP 构建警告。
